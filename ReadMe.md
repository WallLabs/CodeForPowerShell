# CodeForPowerShell
A collection of modules, scripts and other useful files to help with common PowerShell development tasks. 

## Required Tools
* [Visual Studio](https://visualstudio.microsoft.com/) - To build, test and debug.
* [PowerShell Pro Tools](https://github.com/ironmansoftware/powershell-pro-tools) - Support for PowerShell projects and testing within Visual Studio (not provided by Microsoft).
